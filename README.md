Минимальный Telegram-бот для командной работы с задачами.

Как использовать бота:
/start - Начало работы, показывает приветствие и клавиатуру
/add - Добавить задачу:
Бот попросит ввести текст задачи
Введите текст и отправьте
Бот подтвердит добавление
/list - Показать все задачи в чате
/list_csv - Получить файл CSV со всеми задачами
